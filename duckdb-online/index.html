DuckDB online for ad-hoc SQL over files

FileSQL is a standalone DuckDB online tool for querying local files in the browser first and using a native worker only when the workflow truly needs it.

  • CSV, TSV, JSON, JSONL, Parquet, Arrow, and SQLite support.
  • Monaco SQL editor, schema insights, and EXPLAIN modes.
  • Bounded exports, share links, and sample datasets.
