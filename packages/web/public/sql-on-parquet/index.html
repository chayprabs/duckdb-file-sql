SQL on Parquet without a database setup

FileSQL targets ad-hoc Parquet analysis with DuckDB SQL, sample datasets, schema inspection, EXPLAIN plans, and transparent escalation once file sizes exceed the in-browser budget.

  • Parquet queries stay local in the browser when practical.
  • Large Parquet files route to a native DuckDB worker.
  • Exports are available as CSV, JSON, Parquet, and Arrow.
