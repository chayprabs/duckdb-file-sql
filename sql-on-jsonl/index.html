DuckDB SQL on JSONL files

FileSQL loads JSONL fixtures into DuckDB, infers structure, and gives you a SQL editor, schema panel, result paging, and read-only execution rules for exploratory work.

  • JSONL ingestion through DuckDB `read_json_auto`.
  • Autocomplete and sample values after load.
  • Remote URLs can route to the worker when needed.
