SQLite online query with DuckDB

FileSQL supports SQLite sources through DuckDB `sqlite_scanner`, giving you ad-hoc SQL over SQLite files alongside CSV, JSON, Parquet, and Arrow sources in the same playground.

  • Logical tables appear in the schema panel after load.
  • Read-only SQL enforcement blocks mutating statements.
  • Worker mode remains available for larger and remote workloads.
